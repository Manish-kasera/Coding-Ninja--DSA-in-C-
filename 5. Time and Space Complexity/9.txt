Time Complexity of Code

What will be the Time Complexity of following code in terms of ‘n’ ?
Note : Assume k to be a constant value

  for(int i = 0; i < n; i++){
     for(int j = 1 ; j < k; j++){
        cout << (i + j ) << endl;
      }
  }

Options
This problem may have one or more correct answers
O(n^2)
O(n) 
O(logn)
None of these
